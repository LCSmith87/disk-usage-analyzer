# Security Policy

## Supported Versions

The latest stable release and current nightly prereleases receive security fixes.

## Reporting a Vulnerability

Please do not open public issues for suspected vulnerabilities. Report security
concerns through GitHub private vulnerability reporting for this repository, or
email the maintainer if private reporting is unavailable.

Include:

- Affected version or commit.
- Operating system and installer type.
- Steps to reproduce.
- Impact and any known workaround.

The maintainer will acknowledge reports as quickly as practical and coordinate a
fix before public disclosure when the issue is valid.
